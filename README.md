## 使用方式

### clone 至本地

```
git clone https://github.com/wuyuanaaa/blog-server.git
```

### 安装依赖

```
cd blog-server
npm install
```

### 启动服务

__需要预先配置本地的 mongodb 服务器__


> 这里推荐使用 nodemon 进行热启动

```
node bin/www
```
